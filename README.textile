h1. CakePHP Project Skelton with twitter bootstrap

PHP 5.2+, CakePHP 2.0+

Copyright 2011, MontBlancSucks ( https://github.com/MontBlanc-Sucks )

h2. Installation

<pre>
git clone git://github.com/MontBlanc-Sucks/CakePHP-bootstrap-skel.git
</pre>

h2. Usage

<pre>
cd project_name
./Cake/Console/cake bake project app --skel ../CakePHP-bootstrap-skel/skel --empty true

cd app/webroot/css
cp ../../../../CakePHP-bootstrap-skel/bootstrap/bootstrap.min.css .

cd ../js
cp -R ../../../../CakePHP-bootstrap-skel/bootstrap/js ./bootstrap
</pre>
